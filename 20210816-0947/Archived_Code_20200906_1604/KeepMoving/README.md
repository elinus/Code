# Keep Moving

#### 5 steps to beat procrastination

```
1. Just get going. Don't put work off until later.
2. Block out any distractions.
3. Set a timer for 25 minutes.
4. Now focus!
5. When the timer goes off reward yourself!
* Don't worry about finishing your task in this time - you might or you might not.
  The most important thing is to focus!
```
